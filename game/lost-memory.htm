这是一个关于如何找回记忆的故事。

金发蓝眼的年轻人从黑暗中苏醒，发现自己已经失去了所有的记忆，他不知道自己是谁、来自哪里、如今又身居何处。
当他 睁开眼睛的时候，他会看到怎样的景象？

面临选择的时候，他将何去何从？
他能够找回他的过去吗？
对于这位年轻人而言，被遗失的记忆和过去的生活固然重要，但……真的就没有比这两者还要重要的存在了吗？

当最后的结局展现在眼前，我们的主人公是会悲伤、懊悔还是会快乐、欣慰？

当然，这一切都取决于这个游戏的操纵者，也就是——你。
